# QR code component — Frontend Mentor

![Design preview for the QR code component coding challenge](./preview.jpg)

## Описание

Простой и аккуратный компонент QR-кода, собранный в рамках задания с сайта **Frontend Mentor**. Цель проекта — воссоздать дизайн из папки `design/`, соблюдая адаптивность и точность стилей.

**Технологии:** `HTML5`, `CSS3` (Flexbox, Grid), шрифт `Outfit`.

---

## Демонстрация

- **Локально:** откройте `index.html` в браузере или используйте расширение VS Code Live Server.
- **Live demo:** добавьте ссылку на опубликованную версию (GitHub Pages / Vercel / Netlify).

---

## Как запустить проект локально ✅

1. Клонируйте репозиторий:

```bash
git clone <your-repo-url>
```

2. Откройте папку проекта и запустите `index.html` в браузере.
3. (Опционально) Для удобной разработки установите и запустите Live Server в VS Code.

## Структура проекта

- `index.html` — разметка
- `style.css` — основные стили
- `/images` — изображения и ассеты
- `/design` — файлы макетов (мобильный и десктопный)
- `style-guide.md` — подсказки по цветам и шрифтам
